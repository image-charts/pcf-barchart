# Changelog

## [1.0.0] - 2026-05-04

### Added
- Initial release of Image-Charts Bar Chart Generator PCF component
- Support for Enterprise mode (HMAC-SHA256 signing)
- Support for Private Cloud mode (custom domain)
- Vertical and horizontal orientation
- Custom colors and labels
- Debounced updates (300ms)
- Retry with exponential backoff (15s timeout)
- Error placeholder support
- Debug URL display
